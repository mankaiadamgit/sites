Les images et logos proviennent du site "Flaticon" et sont donc la propriété des auteurs qui les ont postés sur le site.

Les prompts utilisés sont les suivants : 
- Génère un arrière plan dans un style coloré, un peu simpliste, sans présence humaine de l'espace (violet - bleu - jaune)
- Comment faire pour avoir un bouton sur son site qui change la feuille de style css ? 
- Comment faire en sorte d'avoir un css différent pour une page en fonction de si elle est mobile ou desktop ?
- W3C me dit aussi The frameborder attribute on the iframe element is obsolete. Est-ce normal ?
- Est-il possible de faire un formulaire qui envoie un mail sans PhP ?
- Peux tu me générer un fond d'écran comme celui ci mais en mode clair ? (image background.png)
- Pour faire un "css alternatif" est-ce qu'on a besoin d'un 2eme fichier css ou pas ?
